About this site

JSComplexity.org runs on Node.js and Express, using Hogan-Express for template rendering and Stylus to generate CSS. The source code is available on GitHub, under the MIT license. The styling was done by genius front-end web developer extraordinaire, Perry Harlock.

The complexity reporting is performed by escomplex, a Node.js-based library that analyses syntax trees generated by the Esprima JavaScript parser. If you need to analyse a large codebase, you might find it easier to install complexity-report, a command-line wrapper for escomplex, on to a development machine or CI server. Doing so will also enable you to take advantage of further complexity metrics for first-order density, change cost and core size, on a project-wide basis. Detailed instructions for setting up complexity-report can be found in the readme file.
